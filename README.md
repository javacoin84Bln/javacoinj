# JavacoinJ Rpc client

 RPC client for Javacoin.
 

# Prerequisites

 Java 8+
 
 maven 3.X.X
 
 
# Installing 
 
``` 
mvn clean install

```

 will generate javacoinj.jar in target folder
 
# Examples
 
  Calling the rpc methods examples are provided in package com.javacoin.examples
